# research/ — Mythos y Glasswing

Working files for the talk. Per-presentation, not shared. After this talk ships, retrospect on what here would have been worth promoting to a cross-presentation wiki (decision deferred per design discussion).

## Files

- **`audience.md`** — who's in the room, what they believe, what they leave with, what to assume vs. establish, expected objections, success criteria.
- **`thesis.md`** — the talk's argument in one sentence + one paragraph + the load-bearing idea.
- **`sources.md`** — every source with tier (T1 primary → T5 unverified), source ID, summary.
- **`quotes.md`** — verbatim quotes only, with source ID + location. Never paraphrase here.
- **`claims.md`** — every claim the talk will make, linked to source IDs, with confidence + counter-evidence flag.
- **`beat-sheet.md`** — narrative arc as numbered beats, before slides.md exists.
- **`open-questions.md`** — manual-verify checklist (bloqueantes + no bloqueantes + logística).

## Discipline

1. Nothing on a slide unless it appears in `claims.md`.
2. Every claim in `claims.md` must link to ≥1 source in `sources.md`.
3. Verbatim text on slides must come from `quotes.md` (with source ID).
4. Update `beat-sheet.md` before touching `slides.md`.
5. Items in `open-questions.md` block their dependent slides until resolved.
